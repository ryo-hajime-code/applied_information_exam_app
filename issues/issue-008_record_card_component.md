# Issue-008: RecordCardコンポーネントの実装

## 概要
`src/components/RecordCard.tsx` に、記録一覧画面で1件の演習記録を表示するカードコンポーネントを実装する。
日付・問題数・正答率・前回比を表示し、削除ボタンを持つ。

## タスク
- [ ] `src/components/RecordCard.tsx` を作成する
  - Props: `RecordCardProps`（`record`, `comparison`, `onDelete`）を使用
  - 表示内容: 日付、「X問中Y問正解」、「正答率: Z%」、前回比
  - 前回比の表示ルール: 正(緑 #34C759 + ⬆)、負(赤 #FF3B30 + ⬇)、ゼロ(グレー #8E8E93 + →)
  - `comparison === null` の場合は「--」を表示（最古の記録）
  - 削除ボタン（ゴミ箱アイコン）を右上に配置し、`onDelete(record.id)` を呼び出す
- [ ] カードスタイルを実装する
  - 背景: #FFFFFF、影: 軽いドロップシャドウ、パディング: 16px、角丸: 12px、マージン: 上下8px
- [ ] 削除後のフェードアウトアニメーション(0.3秒)を実装する（CSSクラスのトグルで対応）
- [ ] 削除ボタンの最小タッチターゲット: 44px × 44px

## 完了条件
- 記録の全情報（日付・正解数・正答率・前回比）が正しく表示される
- 前回比が正負ゼロに応じて色分け表示される
- 削除ボタンをクリックすると `onDelete` が呼ばれる
