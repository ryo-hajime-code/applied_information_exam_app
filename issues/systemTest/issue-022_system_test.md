# Issue-022: システムテスト

## 概要

作成したアプリが要件を満たすかを、ローカル環境にて実際に動かすことでテストする。

---

## テストケース

### 表示系
- 表示されるべき要素が過不足なく表示されている
- パスが / のとき、Homeが表示される
- パスが /records のとき、RecordListが表示される
- 上記以外のパスのとき、/ にリダイレクトされHomeが表示される
- 記録が 0 件のとき、必要な要素が過不足なく表示されている
- 記録があるとき、必要な要素が過不足なく表示されている

### 画面操作系
- 記録を見るボタンクリックで/records へ遷移し、RecordListが表示される
- ホームに戻るボタン押下で/ に遷移し、Homeが表示される
- 受験日を設定できる
- 記録を入力して保存することで、記録が保存される
- 削除ボタンで記録を削除できる
