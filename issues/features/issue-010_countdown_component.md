# Issue-010: CountDownコンポーネントの実装

## 概要
`src/components` に受験日カウントダウン表示と受験日設定UIを一体化した `CountDown.tsx` を実装する。
受験日が設定済みの場合はカウントダウン表示、未設定の場合は設定を促す表示を行い、タップでインライン編集UIに切り替わる。

## タスク
- [ ] `src/components/CountDown.tsx` (または `src/pages/Home.tsx` 内コンポーネント) を作成する
  - Props: `CountDownProps`（`examDate`, `onExamDateChange`）を使用
- [ ] 受験日設定済みの表示を実装する: 「試験まであと X日 [✏️]」
- [ ] 受験日未設定の表示を実装する: 「受験日を設定する →」
- [ ] タップ・編集アイコンクリックでインライン展開する日付入力UIを実装する
  - 日付ピッカー（`<input type="date">`）
  - 「保存」「キャンセル」ボタン
  - 保存時に `validateSettings()` でバリデーション実行
  - バリデーションOKの場合は `updateSettings()` を呼び、`onExamDateChange()` を呼ぶ
- [ ] カウントダウンの日数計算を実装する（`calcDaysUntil()` を使用）
- [ ] 「0日」または「期日超過」の場合の表示ハンドリングを実装する

## 完了条件
- 受験日が未設定の場合に「受験日を設定する」が表示される
- タップでインライン日付入力UIが展開される
- 受験日を保存するとカウントダウン表示に切り替わる
- 編集アイコンタップで再度編集できる
