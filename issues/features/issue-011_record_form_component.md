# Issue-011: RecordFormコンポーネントの実装

## 概要
`src/pages/Home.tsx` 内（または `src/components/RecordForm.tsx` として）、演習記録の入力フォームコンポーネントを実装する。
実施日・問題数・正答数の入力と、リアルタイム正答率計算・インラインバリデーション表示を行う。

## タスク
- [ ] `src/components/RecordForm.tsx` (または `Home.tsx` 内) を作成する
  - Props: `RecordFormProps`（`onSubmit`）を使用
- [ ] 実施日入力フィールドを実装する（`<input type="date">`、デフォルト: 今日の日付）
- [ ] 解いた問題数の `NumberInput` を実装する（placeholder: 例: 80、min: 1、max: 200）
- [ ] 正答した問題数の `NumberInput` を実装する（placeholder: 例: 60、min: 0）
- [ ] リアルタイム正答率表示を実装する（両方入力されるまで「---%」と表示）
- [ ] インラインバリデーションを実装する（入力変更時にエラーを更新）
  - `validateRecord()` を使用
  - エラー時: フィールド直下に赤文字、フィールドを赤枠でハイライト
- [ ] 「記録する」ボタン（プライマリ）を実装し、`onSubmit` コールバックを呼ぶ
- [ ] 送信成功後にフォームをリセットする（日付は今日のまま、数値は空に）

## 完了条件
- 実施日・問題数・正答数を入力し「記録する」ボタンを押すと `onSubmit` が呼ばれる
- 正答率がリアルタイムで計算・表示される
- バリデーションエラーがフィールド直下に表示される
- 送信後にフォームがリセットされる
